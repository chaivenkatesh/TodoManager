# ToDoManager
This Project is to create a ToDO List for user, user can add items to it and delete items from it. Data is stored and managed in backend memory.
Front end is created in angular version 20, node version 22. and backend is using dot net 8.

NOTE: Backend is running in port 7032, if your back end is running in any other port please change port in below file:
https://github.com/chaivenkatesh/TodoManager/blob/master/todo-ui/src/app/services/todoservice.ts

Line 10, private apiUrl = 'https://localhost:7032/api/todos';  Replace localhost:7032 with localhost:your port number
